# AI聊天界面使用说明

## 🎯 界面布局

### 顶部导航栏
- **AI助手标题**：显示"景德镇AI文旅助手"
- **新建会话按钮**：点击开始新的对话
- **查看历史按钮**：显示/隐藏历史记录侧边栏

### 主聊天区域
- **欢迎界面**：首次进入时显示的欢迎信息
- **消息列表**：显示用户和AI的对话内容
- **推荐产品**：AI推荐的产品卡片
- **反馈按钮**：对推荐结果进行评价

### 底部输入区域
- **文本输入框**：输入您的需求
- **发送按钮**：发送消息给AI
- **使用提示**：显示输入建议

## 🚀 使用方法

### 1. 开始对话
1. 在底部输入框中输入您的需求
2. 点击"发送"按钮或按回车键
3. 等待AI分析和推荐

### 2. 输入示例
```
# 景点推荐
"我想找一个适合家庭游玩的景点"
"推荐一些适合拍照的景点"
"我想找一个价格实惠的景点"

# 酒店推荐
"推荐一些性价比高的酒店"
"我想找一个位置方便的酒店"

# 美食推荐
"推荐一些当地特色美食"
"我想找一个适合聚餐的餐厅"

# 陶瓷体验
"推荐一些陶瓷制作体验活动"
"我想学习陶瓷制作"
```

### 3. 查看推荐结果
- AI会分析您的需求
- 推荐3-5个最合适的产品
- 显示推荐理由和产品详情
- 点击产品卡片可查看详情

### 4. 反馈评价
- 对推荐结果点击"有帮助"或"没帮助"
- 帮助AI持续优化推荐质量

### 5. 管理会话
- **新建会话**：点击"新建会话"按钮开始新对话
- **查看历史**：点击"查看历史"查看之前的推荐记录
- **重新查看**：点击历史记录项重新查看推荐结果

- 支持桌面端和移动端
- 自适应不同屏幕尺寸
- 触摸友好的交互设计

### 实时交互
- 实时显示AI思考状态
- 流畅的消息动画效果
- 即时反馈和状态更新

### 产品展示
- 精美的产品卡片设计
- 清晰的价格和位置信息
- 直观的推荐理由展示

### 智能推荐
- 基于AI大模型理解用户需求
- 个性化推荐算法

### 历史管理
- 自动保存推荐历史
- 支持历史记录查看
- 快速重新查看推荐

### 用户反馈
- 简单的反馈收集
- 帮助优化推荐质量

## 📱 移动端使用

### 界面适配
- 历史记录侧边栏在移动端会全屏显示
- 输入框在移动端会自适应键盘高度
- 产品卡片在移动端会单列显示

### 操作优化
- 支持触摸滑动查看历史
- 优化的按钮大小适合手指操作
- 简化的界面布局减少误操作

## 🎯 使用技巧

### 1. 描述需求
- 尽量详细描述您的需求
- 可以包含预算、人数、偏好等信息
- 例如："我想找一个适合家庭游玩的景点，价格不要太贵，有3个大人2个小孩"

### 2. 查看推荐
- 仔细阅读AI的推荐理由
- 点击产品卡片查看详细信息
- 对比不同产品的特点

### 3. 提供反馈
- 及时对推荐结果进行反馈
- 帮助AI了解您的偏好
- 提高后续推荐的准确性

### 4. 管理历史
- 定期查看历史记录
- 重新查看有用的推荐
- 清理不需要的历史记录

## 🚨 注意事项

### 网络要求
- 需要稳定的网络连接
- AI推荐需要调用外部API
- 网络不稳定时可能影响响应速度

### 使用建议
- 一次只发送一个需求
- 等待AI回复后再发送下一条消息
- 避免发送与文旅无关的内容

### 隐私保护
- 您的查询内容会保存到数据库
- 仅用于改善推荐质量
- 不会泄露给第三方

## 🎉 开始使用

1. **登录系统**：确保您已登录用户账号
2. **进入AI助手**：点击首页的"AI智能推荐"或导航栏的"AI助手"
3. **开始对话**：在输入框中输入您的需求
4. **获得推荐**：查看AI推荐的产品和理由
5. **提供反馈**：对推荐结果进行评价

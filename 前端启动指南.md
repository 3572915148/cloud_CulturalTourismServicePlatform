# 前端项目启动指南

## 📦 快速开始

### 1. 安装依赖

打开终端，进入前端项目目录并安装依赖：

```bash
cd /Users/shirenan/Desktop/Zsq_JingdezhenCulturalTourismServicePlatform/frontend
npm install
```

> **首次安装说明**：
> - 安装过程可能需要几分钟，请耐心等待
> - 如果遇到网络问题，可以使用淘宝镜像：`npm install --registry=https://registry.npmmirror.com`

### 2. 启动开发服务器

安装完成后，启动开发服务器：

```bash
npm run dev
```

启动成功后，你会看到类似以下的输出：

```
VITE v5.0.4  ready in 1234 ms

  ➜  Local:   http://localhost:3000/
  ➜  Network: use --host to expose
  ➜  press h to show help
```

### 3. 访问应用

在浏览器中打开：**http://localhost:3000**

## 🚀 功能演示

- 展示平台 Banner 轮播

   - 登录成功后，右上角显示用户信息

### 产品浏览
1. **产品列表**
   - 点击导航栏"文旅产品"
   - 可按分类、区域筛选
   - 支持关键词搜索
   - 分页浏览产品

2. **产品详情**
   - 点击任意产品卡片
   - 查看产品详细信息
   - 查看产品图片、价格、评分
   - 可以立即预订（需登录）

### 陶瓷文化
- 点击导航栏"陶瓷文化"
- 按分类浏览陶瓷相关内容
- 点击卡片查看详细内容

### 个人中心
- 登录后点击右上角用户头像
- 选择"个人中心"
- 可管理：
  - 个人信息
  - 我的订单
  - 我的收藏
  - 我的评价

  - 账号设置

## 🔧 开发说明

### 项目结构

```
frontend/
├── src/
│   ├── api/          # API接口
│   ├── router/       # 路由配置
│   ├── stores/       # 状态管理
│   ├── utils/        # 工具函数
│   ├── views/        # 页面组件
│   ├── App.vue       # 根组件
│   └── main.js       # 入口文件
├── public/           # 静态资源
├── index.html        # HTML模板
├── package.json      # 依赖配置
└── vite.config.js    # Vite配置
```

### 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 下一代前端构建工具
- **Element Plus** - Vue 3 UI 组件库
- **Vue Router** - Vue.js 官方路由
- **Pinia** - Vue 状态管理库
- **Axios** - HTTP 客户端

### 端口说明

- 前端开发服务器：`http://localhost:3000`
- 后端API服务器：`http://localhost:8080`
- 前端请求会自动代理到后端（已在vite.config.js中配置）

## ⚠️ 注意事项

### 1. 后端服务必须先启动

前端依赖后端API，请确保后端服务已在 `http://localhost:8080` 正常运行。

**检查后端是否启动：**
```bash
# 在浏览器访问
http://localhost:8080/api
```

如果能看到返回数据或错误信息（非连接拒绝），说明后端已启动。

### 2. 首次使用需要注册

- 数据库是空的，需要先注册一个账号
- 注册成功后可以登录使用

### 3. 产品数据

- 初始状态下可能没有产品数据
- 可以通过后端API添加测试数据
- 或运行后端项目中的 `schema.sql` 初始化数据

### 4. 图片显示

项目中使用了在线图片服务（Unsplash），需要：
- 网络连接正常
- 如果图片加载慢或失败，这是正常的网络延迟

### 5. 浏览器兼容性

推荐使用以下浏览器：
- Chrome（推荐）
- Edge
- Firefox
- Safari

## 🐛 常见问题

### Q1: npm install 失败

**解决方案：**
```bash
# 清理缓存
npm cache clean --force

# 使用淘宝镜像
npm install --registry=https://registry.npmmirror.com
```

### Q2: 端口 3000 被占用

**解决方案：**
1. 修改 `vite.config.js` 中的端口：
```javascript
server: {
  port: 3001,  // 改为其他端口
  // ...
}
```

### Q3: API 请求失败

**可能原因：**
1. 后端服务未启动
2. 端口配置不正确

**解决方案：**
1. 确保后端在 8080 端口运行
2. 检查 `vite.config.js` 中的代理配置

### Q4: 页面样式错乱

**解决方案：**
```bash
# 重新安装依赖
rm -rf node_modules package-lock.json
npm install
```

### Q5: 热更新不工作

**解决方案：**
```bash
# 停止开发服务器（Ctrl+C）
# 重新启动
npm run dev
```

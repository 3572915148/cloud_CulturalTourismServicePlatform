## 🚀 项目启动指南

### 1. 环境要求
- Java 8+
- Node.js 16+
- MySQL 8.0+
- Maven 3.6+

### 2. 启动步骤
```bash
# 1. 启动后端
cd backend
mvn spring-boot:run

# 2. 启动前端
cd frontend
npm install
npm run dev
```

### 3. 访问地址
- 用户端：http://localhost:3000
- 商户端：http://localhost:3000/merchant/login
- 管理员：http://localhost:3000/admin/login
